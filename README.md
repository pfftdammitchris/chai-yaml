# `chai-yaml`

> YAML Assertions for Chai

## Usage

```
const chai = require('chai')
const chaiYaml = require('chai-yaml');

chai.use(chaiYaml)
```

## Peer/Dev Dependencies

| Name                                   | Version  |
| -------------------------------------- | -------- |
| [chai](https://github.com/chaijs/chai) | `^4.6.0` |
| [yaml](https://github.com/eemeli/yaml) | `^2.1.0` |
